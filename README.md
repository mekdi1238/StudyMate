# StudyMate

## Project Description

StudyMate is an AI-powered study assistant that helps students convert educational content into organized study materials.

Users can submit a topic and source text, then generate:

* Study notes
* Practice quizzes
* Answer keys

The application is built using:

* HTML
* CSS
* JavaScript
* EJS
* Node.js
* Express.js
* PostgreSQL
* Google Gemini API

## Features

* Create study sets
* View saved study sets
* Edit study sets
* Delete study sets
* Generate notes using AI
* Generate quizzes using AI
